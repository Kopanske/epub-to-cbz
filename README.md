# epub-to-cbz
Tool to convert a ebook to a comic book

## Requirements
* Java runtime 21 or higher

## Usage
java -jar ePubToCbz-1.0.0-all.jar [input directory] [output directory]

Example:

java -jar ePubToCbz-1.0.0-all.jar c:\myEbooks c:\myComicBooks

## Release
To create a new release create a tag and push it to Github.

Example:

git tag v1.0.0

git push --tags
